## Cài đặt & Chạy

```bash
# Clone repository
git clone https://github.com/tkhoaaa/portfolio-vtk.git
cd portfolio-vtk

# Cài đặt dependencies
npm install

# Chạy development server
npm run dev

# Build production
npm run build

# Preview production build
npm run preview
```
